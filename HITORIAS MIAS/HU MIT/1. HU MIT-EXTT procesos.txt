INTEROPERABILIDAD
********************

1. GUARDAR BK DE LOS OBJ ANTERIORES (BKLUZTVQ/BODEGATMP)
2. PONES LOS NUEVOS
3. GUARDAR BK A ESTE ARCHIVO TANDEM.PTLF_TVQ (archivo insumo de interoperabilidad)
4. LIMPIAR MARCAS DEL ARCHIVO (UPDATE TANDEM/PTLF_TVQ                       
SET PTLEST = ''   , PTLFEC = 0 ,  PTLHOR = 0)
5. AMBIENTAR AMB
6. LIMPIAR ARCHIVOS: 
   DELETE ELECTRON CEF0300A
7. DELETE SWITCH SWFOPERA
8. CALL PGM(LGOMEZ/ADQBORRA)
9. UPDATE   BASEDOS.VQFCNTR0  SET DCNACT ='P'
10. CALL PGM(BASEDOS/VQC0100) 
11. PASO 1 Y 2
12. PASO DOS ME PREGUNTA SI ES DEFINITIVO DIGO: N  
112.1 CPYF FROMFILE(TANDEM/PTLF_TVQ) TOFILE(SWITCH/SWFEXTRA) MBROPT(*REPLACE)  
FMTOPT(*NOCHK)     
13. CONSULTAR CEF0300A SI LOS CAMPOS DE CH ESTAN VACIOS
14. PARA QUE SE ME GENERE EL EXTRACTO, ES LO SIGUIENTE: 
15. MIRAR FECHA DE CANJE SI LO EJECUTE HOY DEBE TENER FECHA DE MAÑANA: 
SELECT * FROM CANJE/CAFCNTRL  WHERE TANOCA =' 24' AND TMESCA =' 05' AND TDIACA = '25' ORDER BY TFECHA desc
16. SI NO ESTA LA FECHA BIEN:
UPDATE  BODEGATMP/CAFCNTRLYS  SET TANOCA=24, TMESCA=05, TDIACA=25, TFECHA=240525
CPYF FROMFILE(BODEGATMP/CAFCNTRLYS) TOFILE(CANJE/CAFCNTRL) MBROPT(*ADD)

CHGJOB DATE(250524) FECHA DE CANJE OSEA DE MAÑANA

17. AMBIENTO AMB CALL BASEDOS/CAC0700S
18. LLAMO EL PROGRAMA DE DON JORGE PARA MIRAR LOS EXTRACTOS PARA QUE ME SALGAN ESTOS TRES EXTRACTOS:
CALL PGM(ELECTRON/CEC1840) PARM('240525' 'N' 'N' '0051' '0001' '0013' '0051' '0001' '0013')
19. APARECE ESTE EROOR:
El procedimiento GASFTPGET ha recibido GAE1003. (C D I R) SI SALE ESTE MENSAJE RESPONDER  *I*  --- SI QUIERO DOY WRKACTJOB PARA MIRAR MENSAJES I 
20.EL AL FINAL NO SE PORQUE BOTA DSPY ERROR DSPY ERROR DSPY ERROR DSPY ERROR
21. SHIFT + ESC ... ESCRIBIR 2
22. ENTER
23. Y REVISAR LOS EXTRACTOS DE LA TD0051 TDD0001 Y TD0013:

, A.* FROM TD0051/EXTRACTO A
